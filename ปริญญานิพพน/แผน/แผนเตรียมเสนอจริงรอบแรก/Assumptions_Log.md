## วิธีใช้

| คอลัมน์ | ความหมาย |
|---|---|
| **สมมติฐาน** | ชื่อตัวแปร/ข้อสมมติ (เช่น MHR, ราคามีดใหม่, อายุมีดเฉลี่ย) |
| **ค่า** | ตัวเลขหรือช่วงที่ใช้ |
| **แหล่ง** | บัญชี / ช่างลับ / ข้อมูลเก่า / สมมติผู้วิจัย |
| **วันที่** | วันที่บันทึกหรืออัปเดตล่าสุด |
| **สถานะ** | `verified` = ยืนยันจากแหล่งทางการแล้ว | `unverified` = ยังรอ / ใช้ sensitivity |
