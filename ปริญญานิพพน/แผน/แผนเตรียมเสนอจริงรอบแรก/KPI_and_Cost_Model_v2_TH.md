## 2. แบบจำลองต้นทุน (สมการ + สมมติฐาน)

### 2.1 ตัวแปรต้นทุน
- `MHR` = Machine-Hour Rate (บาท/ชม.) — **ต้องขอจากฝ่ายบัญชี** (ปัจจุบันรู้แค่ค่าแรง 50 บาท/ชม.)
- `Cp` (เปลี่ยนตามแผน) = ค่าลับมีด + (เวลาเปลี่ยนตามแผนหลัง SMED ÷ 60) × MHR
- `Cf` (มีดพังกลางคัน) = ค่ามีดใหม่ + (Downtime ฉุกเฉิน ÷ 60) × MHR + (Scrap ต่อชิ้น × จำนวนชิ้นเสียเมื่อพัง)

### 2.2 จุดเปลี่ยนคุ้มสุด (Age-Replacement)
\[ C(t_p) = \frac{C_p R(t_p) + C_f [1-R(t_p)]}{\int_0^{t_p} R(t)\,dt},\quad R(t)=e^{-(t/\eta)^\beta} \]
หา \(t_p^*\) เชิงตัวเลข (ดู `age_replacement_optimization.py`)

### 2.3 Net Saving / ปี / เครื่อง
\[ \text{Saving/ปี} = (CPP_{baseline} - CPP_{t_p^*}) \times \text{ผลผลิตต่อปี} \]
ผลผลิตต่อปี ≈ 2,000 ชิ้น/วัน × วันทำงาน/ปี (ระบุจำนวนวันจริง)

## 3. ตัวอย่างเชิงสาธิต (ILLUSTRATIVE — ห้ามอ้างในเล่มจนกว่าจะมีข้อมูลจริง)

> ใช้ค่าสมมติใน [04_Cost_Baseline_Inputs.csv](../รายงาน/ข้อมูลประกอบ/templates/04_Cost_Baseline_Inputs.csv) (MHR=450, มีดใหม่=250, ค่าลับ=80, downtime พัง=30 นาที, scrap=40฿×20 ชิ้น) และ β=3.1, η=14,000

- `Cp` ≈ 80 + (5/60)×450 ≈ **117.5 บาท**
- `Cf` ≈ 250 + (30/60)×450 + (40×20) ≈ **1,275 บาท**
- ผลลัพธ์ `t_p*` และ CPP ให้รันสคริปต์เพื่อยืนยัน (ตัวเลขจะเปลี่ยนตามอินพุต)

**ข้อสรุปเชิงสาธิต:** มูลค่าประหยัดต่อปีจากการลด downtime มีดโดยตรง "ไม่มาก" — มูลค่าหลักมาจาก **การกัน scrap ทองเหลืองเมื่อมีดพังกลางคัน** และการวางแผนที่แม่นขึ้น

## 4. การนำเสนออย่างซื่อสัตย์ (Reviewer-proof)
1. ทุกตารางผลลัพธ์ระบุ "ILLUSTRATIVE" จนกว่าจะแทนค่าจริง
2. ทำ **Sensitivity Analysis** ของ `Cf` และ `MHR` (สคริปต์รองรับ) เพื่อแสดงความคงทนของ `t_p*`
3. แยกชัดเจนระหว่าง "เงินประหยัดจริง (realized)" กับ "การหลีกเลี่ยงต้นทุนแฝง (avoided cost)"
4. ระบุข้อจำกัด: downtime มีดเป็นสัดส่วนเล็กของ downtime รวม; No-demand ไม่ใช่ loss
